# Mcl-ChieriSetu
为Mirai开发的插件--虽然目前只有很少功能，但未来会有更多的更新    
  
    
      
**☆它有什么用？**
  
1.发涩图，发随机涩图（没错）  
  
2.（随机/指定）对某人发送发病小作文（还在写）  
  
**☆特殊功能**

随机踢人 （在群内发送/随机踢人即可，需要[Chat-command](https://github.com/project-mirai/chat-command)插件）  

**☆未来更新计划：**  
  
1.添加更多的可自定义内容    
  
**☆使用方式：**  
  
1.在群内发送  “涩图” 即可
